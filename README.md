## Run Locally

Clone the project

```bash
  git clone https://github.com/IbraheemQadi/Portfolio.git
```

Go to the project directory

```bash
  cd Portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
